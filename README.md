# Тестовое задание — Усманова

Демо-сборка 3 экранов лендинга https://usmanovafit.gymteam.ru/mainpage.

**Live:** https://player2427.github.io/usmanova-testovoe/

## Что собрано

1. **Hero** — главный экран с заголовком, подзаголовком, двумя CTA, портретом, «живым» бейджем «Тренируются прямо сейчас» и блоком статистики.
2. **Программы** — секция с карточками тренировок. На десктопе — сетка 3×2, на планшете — 2 в ряд, на мобильном — горизонтальный scroll-snap-карусель.
3. **Помощь с выбором** — тёмная секция с рабочей формой подбора программы (имя, телефон/Telegram, цель, согласие).

Дополнительно: модальное окно «Подобрать программу», валидация полей, маска телефона, toast-уведомления.

## Инструмент

Vanilla HTML + CSS + JS, без сборщика. Шрифт — Manrope (Google Fonts). Стили `clamp()` для типографики, `aspect-ratio` для фото, `scroll-snap` для мобильной карусели. Совместимо со статическим хостингом GitHub Pages.

## Время

≈ 1.5 часа.

## Запуск локально

Открыть `index.html` в браузере — никакой сборки не нужно.
